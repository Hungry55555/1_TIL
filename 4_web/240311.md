# 반응형 웹
## Bootstrap Grid system
웹페이지 레이아웃을 조정하는데 사용되는 **12개의 컬럼**으로 구성된 시스템
단순히 나누는게 아닌 레이아웃 조정을 12개컬럼으로 맞춰 지정하겠다는것
* 왜 12개??
  하나의 웹 페이지를 다양한 기기에 적절하게 맞춰서 표현하고 싶은 것
  약수가 많기 때문
  안에들어가서 작은 데도 같은 레이아웃이 적용됨

## Grid system 목적
