상속(Inheritance)

클래스 간의 상하관계


이런 기능이 왜 필요하징??

1. 코드 재사용
    * 상속을 통해 기존 클래스의 속서오과 메서드
    * 코드 중복 제거

2. 계층구조
    * 부모 자식 클래스 관계가 생김
    * 더 구체적인 계층구조 만들수있음

3. 유지 보수의 용이성
    * 기존 클래스에 수정하고 싶다면 하나의 클래스만 수정하면 되기때문에 유지보수 용이
    * 모터 비히클이라는 큰 클래스가 있을때 저걸 기준으로 하이클래스 세개가 만들어짐
